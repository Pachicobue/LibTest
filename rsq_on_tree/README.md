# 木のパス上総和クエリ

## 問題文

`N`頂点の無向木`G`と長さ`N`の数列`C`が与えられます。`Q`個のクエリが飛んでくるので処理結果を出力して下さい。  

- `a_i b_i`: 頂点`a_i,b_i`を結ぶパス上の頂点`v`について`C_v`の総和を出力して下さい。

## 制約

- `G`は木
- `2 <= N <= 100000`
- `2 <= Q <= 100000`
- `-100000 <= C_i <= 100000`
- `0 <= u_i,v_i < N`(辺の端点)
- `0 <= a_i,b_i < N`(クエリ点)

## 入力

```
N Q
C_0 C_1 ... C_{N-1}
u_1 v_1
u_2 v_2
.
.
u_{N-1} v_{N-1}
a_1 b_1
a_2 b_2
.
.
a_Q b_Q
```

## 出力

```
ans_1
ans_2
.
.
ans_Q
```
