# nCk (mod P) の計算(実行時MOD)

## 問題文

素数`P`が与えられます。`Q`個のクエリが飛んでくるので処理して下さい。
- `n k`: `nCk (mod P)`を出力して下さい

## 制約

- `2 <= P <= 1000000000`
- `1 <= Q <= 100000`
- `0 <= k_i,n_i <= 100000`
- `P`は素数

## 入力

```
P Q
n_1 k_1
n_2 k_2
.
.
n_Q k_Q
```

## 出力

```
a_1
a_2
.
.
a_Q
```

## サンプル

入力
```
257 6
33 15
15 14
22 8
17 36
11 33
27 26
```

出力
```
125
15
62
0
0
27
```
