# K-ボナッチ数列のn項目(実行時MOD)

## 問題文

素数`P`が与えられます。正の整数`N,K`が与えられます。以下のように数列`A`を決めるとき、`A_N`を出力して下さい。  

- `A_0=A_1=...=A_{K-1}=1`
- `A_n=(A_{n-1}+A_{n-2}+...+A_{n-K}) (mod P)`

## 制約

- `2 <= P <= 1000000000`
- `1 <= N <= 1000`
- `1 <= K <= 10`
- `P`は素数

## 入力

```
N K P
```

## 出力

```
A_N
```

## サンプル

入力
```
59 5 8053
```

出力
```
5877
```
