# オイラー路

## 問題文

`N`頂点`M`辺の無向グラフ`G`が与えられます。`G`上のオイラー路を出力して下さい。  
オイラー路が存在することは仮定して良いです。

## 制約

- `1 <= N <= 100000`
- `1 <= M <= 300000`
- `0 <= u_i,v_i < N`(辺の端点)
- `G`は連結かつ一筆書き可能

## 入力

```
N M
u_1 v_1
u_2 v_2
.
.
.
u_M v_M
```

## 出力

`G`のオイラー路の通る頂点番号を順に出力して下さい。  
オイラー路が複数考えられる場合は、任意のオイラー路を出力して良いです。
```
a_0 a_1 ... a_M
```
