# 直交領域探索クエリ

## 問題文

長さ`N`の数列`V`が与えられます。`Q`個のクエリか飛んでくるので処理結果を出力して下さい。

- `l r a b`: `V_l,V_{l+1},...,V_{r-1}`の中で`[a,b)`の範囲に含まれるものの個数を出力して下さい

## 制約

- `1 <= N <= 100000`
- `1 <= Q <= 100000`
- `0 <= V_i <= 1000000`
- `0 <= l_i < r_i <= N`
- `0 <= a_i < b_i <= 1000001`

## 入力

```
N Q
V_0 V_1 ... V_{N-1}
l_1 r_1 a_1 b_1
l_2 r_2 a_2 b_2
.
.
l_Q r_Q a_Q b_Q
```

## 出力

```
ans_1
ans_2
.
.
ans_Q
```
