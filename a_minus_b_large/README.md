# A-B (Large Version)

## 問題文

整数 `A,B` が与えられます。 `A-B` を出力して下さい。

## 制約

- `1 <= |A|,|B| < 10^{100000}`

## 入力

```
A B
```

## 出力

```
A-B
```

## サンプル

入力
```
7413 77894663596174135476145693678475404781800572602993688787067902
```

出力
```
-77894663596174135476145693678475404781800572602993688787060489
```
