# K-ボナッチ数列のn項目(固定MOD)

## 問題文

`P=924844033`とします。正の整数`N,K`が与えられます。以下のように数列`A`を決めるとき、`A_N`を出力して下さい。  

- `A_0=A_1=...=A_{K-1}=1`
- `A_n=(A_{n-1}+A_{n-2}+...+A_{n-K}) (mod P)`

## 制約

- `1 <= N <= 1000`
- `1 <= K <= 10`

## 入力

```
N K
```

## 出力

```
A_N
```

## サンプル

入力
```
80 6
```

出力
```
201832363
```
