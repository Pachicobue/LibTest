# 最小全域木のコスト

## 問題文

`N`頂点`M`辺のコスト付き無向グラフ`G`が与えられます。`G`の最小全域木のコストを出力して下さい。

## 制約

- `2 <= N <= 100000`
- `N-1 <= M <= 300000`
- `0 <= u_i,v_i < N`(辺の端点)
- `-100000 <= c_i <= 100000`(辺のコスト)
- `G`は連結

## 入力

```
N M
u_1 v_1 c_1
u_2 v_2 c_1
.
.
u_{N-1} v_{N-1} c_{N-1}
```

## 出力

`G`の最小全域木のコストを出力して下さい。
```
C
```
