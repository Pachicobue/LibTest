# 区間最小値クエリ(更新クエリなし)

## 問題文

長さ`N`の数列`V`が与えられます。`Q`個のクエリか飛んでくるので処理結果を出力して下さい。  

- `l_i r_i`: `V_{l_i}+V_{l_i+1}+...+V_{r_i-1}`を出力して下さい

## 制約

- `1 <= N <= 100000`
- `1 <= Q <= 100000`
- `-1000000000 <= V_i <= 1000000000`
- `0 <= l_i < r_i <= N`

## 入力

```
N Q
V_0 V_1 ... V_{N-1}
l_1 r_1
l_2 r_2
.
.
l_Q r_Q
```

## 出力

```
ans_1
ans_2
.
.
ans_Q
```
