# 橋の列挙

## 問題文

無向グラフ`G`が与えられます。`G`の[橋](http://judge.u-aizu.ac.jp/onlinejudge/description.jsp?id=GRL_3_B&lang=ja)を列挙して下さい。

## 制約

- `G`は連結
- `2 <= V <= 100000`(頂点数)
- `V-1 <= E <= 110000`(辺数)
- `0 <= u_i,v_i < V`(辺の端点)

## 入力

1行目は頂点数`V`,辺数`E`  
2行目から(E+1)行目は辺の情報が `始点 終点` の形式で与えられます。
```
V E
u_1 v_1
u_2 v_2
.
.
u_E v_E
```

## 出力

1行目に橋の本数を出力し、  
2行目に橋`a b`を辞書順で昇順に出力して下さい。  
但し、橋`(a,b)`は`a<=b`を満たすように出力しなければいけません。
```
m
a_1 b_1
a_2 b_2
.
.
a_m b_m
```
