# 中国人剰余定理

## 問題文
`N`個の整数組`(R_i,M_i)`が与えられます。各`i`について、`V = R_i (mod M_i)`を満たすような最小の非負整数`V`を出力して下さい。  
このような`V <= 10^{15}`が存在することは仮定して良いです。

## 制約

- `2 <= N <= 5`
- `1 <= M_i <= 100000`
- `0 <= R_i < M_i`

## 入力

```
N
R_1 M_1
R_2 M_2
.
.
R_N M_N
```

## 出力

```
V
```

## サンプル

入力
```
5
486 864
315 681
248 274
630 717
102 832
```

出力
```
35046
```
